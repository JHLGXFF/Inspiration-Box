# Project Spec: 灵感收集箱 (Inspiration Box)

## 1. 目标
创建一个微型全栈应用，用于快速记录和展示个人灵感。

## 2. 核心功能
- **灵感输入**：支持用户在前端页面输入一段文字或一个 URL 链接。
- **自动记录**：后端在保存时需自动生成 `created_at` 时间戳。
- **标签系统**：用户在创建灵感时可以添加多个标签（Tags）。
- **瀑布流展示**：前端首页以卡片流或瀑布流形式展示所有灵感，按时间倒序排列。

## 3. 数据模型 (Data Model)
- **Memo**:
  - `id`: Integer (Primary Key)
  - `content`: String (Text or URL)
  - `tags`: String (以逗号分隔，或作为关联表)
  - `created_at`: DateTime

## 4. 交互流程
1. 用户访问页面。
2. 用户在顶部的输入框输入内容并添加标签，点击提交。
3. 页面立即刷新或局部更新，下方卡片流出现新的灵感卡片。
